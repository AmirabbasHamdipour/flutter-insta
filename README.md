# Flutter Insta

## Project Overview
Flutter Insta is a social media application built using the Flutter framework. The app allows users to share photos and videos, follow friends, and explore content shared by others in a visually appealing interface. 

## Features
- User Authentication
- Posting photos and videos
- Liking and commenting on posts
- Following and unfollowing users
- Real-time updates
- Explore feed to discover new users and content

## Getting Started
To get a local copy up and running follow these simple steps:

### Prerequisites
- Flutter SDK
- Dart SDK

### Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/AmirabbasHamdipour/flutter-insta.git
   ```
2. Navigate into the project directory:
   ```sh
   cd flutter-insta
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
